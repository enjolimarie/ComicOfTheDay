About

What is this?

Comic of the Day picks a random Marvel comic issue every day from a curated pool of characters — Wolverine, Ghost Rider, Doctor Strange, Venom, Moon Knight, Rogue, Gambit, and members of the X-Men and Fantastic Four. The comic stays the same all day, then changes at midnight.

How it works

  • A character is selected each day using a date-seeded random algorithm.
  • The Marvel API is queried for that character's comics (server-side only — private key never reaches the browser).
  • A valid comic cover is chosen, also seeded by the current date.
  • The cover image is downloaded server-side and two dominant colors are extracted using colorthief.
  • Those colors become the page's gradient background — no canvas CORS issues.
  • Results are cached in a daily JSON file so the Marvel API is called at most once per day.
