API Reference

GET /

Returns the Comic of the Day as an HTML page with dynamic gradient background.

GET /api/comic

Returns today's comic data as JSON. Served from the daily cache after the first request.

Response Fields

Field Type Description
datestringISO date (YYYY-MM-DD)
character_idintegerMarvel character ID used
comic_idintegerMarvel comic ID
titlestringFull comic title
series_namestringSeries name
issue_numbernumberIssue number within series
image_urlstringCover image URL (Marvel CDN)
dominant_colorstringHex color extracted from cover (#rrggbb)
secondary_colorstringSecond distinct hex color from cover

Example Response

{
  "date": "2026-05-21",
  "character_id": 1009718,
  "comic_id": 67890,
  "title": "Wolverine (1982) #1",
  "series_name": "Wolverine (1982)",
  "issue_number": 1,
  "image_url": "https://i.annihil.us/u/prod/marvel/.../portrait_incredible.jpg",
  "dominant_color": "#c0392b",
  "secondary_color": "#2c3e50"
}
